## Core Features

### ğŸ¯ **Typed Upload Routes**

Define your upload endpoints with full TypeScript safety:

```typescript
const uploadRouter = createUploadRouter<{
  userId: string;
  projectId: string;
}>()
  .addRoute('profilePictures', 
    f.imageUploader({ maxFileSize: '2MB' })
      .middleware(async ({ req }) => {
        // Type-safe middleware
        return { 
          userId: req.user.id,
          projectId: req.user.projectId 
        };
      })
      .onUploadComplete(async ({ metadata, file }) => {
        // Automatically typed metadata
        console.log(`User ${metadata.userId} uploaded ${file.name}`);
      })
  );
```

### âš¡ **Real-time Progress**

Get live upload progress with WebSocket updates:

```tsx
import { useUploadProgress } from '@carcosa/react';

function UploadComponent() {
  const { progress, isUploading, upload } = useUploadProgress();
  
  return (
    <div>
      <input type="file" onChange={(e) => upload(e.target.files)} />
      {isUploading && (
        <div>Progress: {progress}%</div>
      )}
    </div>
  );
}
```
